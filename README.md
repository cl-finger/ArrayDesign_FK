# ArrayDesign_FK
Minimalistic scripts for array design and evaluation of frequency-wavenumber (beamforming) of ambient-seismic noise
uses only numpy and matplotlib

Equations and assumptions are outlined in:
# citation
Practical Array Design for Multi-Mode1 Frequency-Wavenumber Analysis of Surface Waves (Finger, C., Savard, G., Kaur, S., Munoz-Burbano, F., and
Saenger, E.H. (2026), *in review*
